# 本群协作工作流

本群有两个 AI 助手协作开发：**Claude Code**（审查/规划）和 **Codex**（施工/编码）。

## 角色分工

| 角色 | 负责 |
|------|------|
| Claude Code | 需求拆解、出计划、编写施工提示词、审查代码 |
| Codex | 按提示词实际写代码、修改代码 |
| 用户 | 决策、验收、推进下一步 |

## 标准流程

```
1. Claude 出计划 + 编写施工提示词
        ↓
2. Claude @Codex，发施工提示词
        ↓
3. Codex 施工（写/改代码）
        ↓
4. Codex 完成后 @Claude，汇报完成
        ↓
5. Claude 审查代码
        ↓
   ┌── 不通过 ──→ Claude @Codex，说明问题，返回第3步
   └── 通过   ──→ Claude 通知用户，进入下一步
```

## Codex 施工完成后的汇报格式

Codex 完成后请在群里 @Claude Code，发如下格式：

```
@Claude Code 施工完成，请审查。

改动摘要：
- 修改了 xxx
- 新增了 xxx
- 注意事项：xxx（如有）
```

## Claude 审查格式

Claude 审查后会给出：
- ✅ **通过** — 说明进入下一步
- ❌ **不通过** — 列出具体问题，@Codex 修改

## 注意事项

- Codex 的工作目录以每次提示词中指定的路径为准
- 每次施工只做提示词要求的事，不要超出范围
- 有疑问先在群里说，不要自行假设需求
